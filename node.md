#

### Mostrar todas as Versões do Node
```
nvm list-remote
```

### Instalando o Node
```
nvm install v18.14.1
```

### Setando uma versão do Node como padrão
```
nvm alias default v18.14.1
```

### Usar uma versão específica do Node temporariamente
```
nvm use v18.14.1
```
